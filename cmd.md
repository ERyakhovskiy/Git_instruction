# Подсказки по командной строке

Команда смена директории
```sh
cd c:\folder_name
```

Кооманда отображения текущей директории
```sh
pwd
```

Листинг текущей директории
Window:
```sh
dir
```
Linux, MacOs:
```sh
ls
``````

удаление файла в Windows:
``````sh
del <filename>
``````
в Linux, MacOs : 
``````sh
rm <filename>
``````
